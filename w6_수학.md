# 코딩테스트지도 6주차 수학 - 소수
## 백준 - 소수 구하기
- 문제
  - M이상 N이하의 소수를 모두 출력하는 프로그램을 작성하시오.

- 입력
  - 첫째 줄에 자연수 M과 N이 빈 칸을 사이에 두고 주어진다. (1 ≤ M ≤ N ≤ 1,000,000) M이상 N이하의 소수가 하나 이상 있는 입력만 주어진다.

- 출력
  - 한 줄에 하나씩, 증가하는 순서대로 소수를 출력한다.


## 문제 풀이 과정과 생각
먼저, 강의자료에 있는 소수 판별과 에라토스테네스의 체를 이해한다. 문제에서 주어진 범위가 1,000,000 까지 이므로 시간 복잡도를 고려하여 에라토스테네스의 체 알고리즘을 활용한다. 
배열에 m 까지의 모든 수를 소수로 초기화 한 후, 2부터 루트 m까지 소수일 때 배수를 제거한다. 


# 코드로 구현
```
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int n = scanner.nextInt();
        int m = scanner.nextInt();

        boolean[] isPrime = new boolean[m+1];

        // 모든 수를 소수로 초기화
        for (int i = 2; i <= m; i++) {
          isPrime[i] = true;
        }

        // 에라토스테네스의 체 구현
        for (int i = 2; i * i <= m; i++) {
          if(isPrime[i]) {   // 소수일 때 배수 제거
            for (int j = i * i; j <= m; j += i) { 
              isPrime[j] = false;
            }
          }
        }

        // 입력받은 수 사이의 소수 판별
        for (int i = n; i <= m; i++) {
          if(isPrime[i]) System.out.println(i);
        }

      scanner.close();
    }
}

```

## 느낀점
문제만 읽었을 때는 매우 간단해 보였으나 에라토스테네스의 체를 구현하는 것은 생각보다 복잡했다. 특히 소수의 배수를 제거하는 부분이 헷갈렸다. 이론을 읽고 이해했다고 생각했으나 직접 구현해보니 생각보다 어려워 구현하는 데 시간이 걸렸다.
