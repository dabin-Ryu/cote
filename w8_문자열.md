# 코딩테스트지도 8주차 문자열
## 프로그래머스 - 신규 아이디 추

- 문제
카카오에 입사한 신입 개발자 네오는 "카카오계정개발팀"에 배치되어, 카카오 서비스에 가입하는 유저들의 아이디를 생성하는 업무를 담당하게 되었습니다. "네오"에게 주어진 첫 업무는 새로 가입하는 유저들이 카카오 아이디 규칙에 맞지 않는 아이디를 입력했을 때, 입력된 아이디와 유사하면서 규칙에 맞는 아이디를 추천해주는 프로그램을 개발하는 것입니다.
다음은 카카오 아이디의 규칙입니다.

아이디의 길이는 3자 이상 15자 이하여야 합니다.
아이디는 알파벳 소문자, 숫자, 빼기(-), 밑줄(_), 마침표(.) 문자만 사용할 수 있습니다.
단, 마침표(.)는 처음과 끝에 사용할 수 없으며 또한 연속으로 사용할 수 없습니다.
"네오"는 다음과 같이 7단계의 순차적인 처리 과정을 통해 신규 유저가 입력한 아이디가 카카오 아이디 규칙에 맞는 지 검사하고 규칙에 맞지 않은 경우 규칙에 맞는 새로운 아이디를 추천해 주려고 합니다.
신규 유저가 입력한 아이디가 new_id 라고 한다면,

1단계 new_id의 모든 대문자를 대응되는 소문자로 치환합니다.
2단계 new_id에서 알파벳 소문자, 숫자, 빼기(-), 밑줄(_), 마침표(.)를 제외한 모든 문자를 제거합니다.
3단계 new_id에서 마침표(.)가 2번 이상 연속된 부분을 하나의 마침표(.)로 치환합니다.
4단계 new_id에서 마침표(.)가 처음이나 끝에 위치한다면 제거합니다.
5단계 new_id가 빈 문자열이라면, new_id에 "a"를 대입합니다.
6단계 new_id의 길이가 16자 이상이면, new_id의 첫 15개의 문자를 제외한 나머지 문자들을 모두 제거합니다.
     만약 제거 후 마침표(.)가 new_id의 끝에 위치한다면 끝에 위치한 마침표(.) 문자를 제거합니다.
7단계 new_id의 길이가 2자 이하라면, new_id의 마지막 문자를 new_id의 길이가 3이 될 때까지 반복해서 끝에 붙입니다.

## 문제 풀이 과정과 생각
강의 시간에 배운 정규식을 활용하여 1-7단계까지의 과정을 차례로 구현한다. 
1단계 : 자바 내장 메서드를 사용해 대문자를 소문자로 치환

2단계 : 알파벳, 숫자, -, _, . 만 남기고 제거
        정규식으로 허용문자 표현 [a-z0-9\\-_.] (특수기호 앞에 \\를 붙인다)
        제외한 나머지 문자를 의미하는 ^ 를 활용        
        
3단계 : 연속된 마침표 하나로 치환
        마침표가 2번 이상 반복 : \\.{2,}
        
4단계 : 처음/끝에 마침표 있을 때 제거
        처음 : ^\\.
        끝 : \\.$
        
5단계 : 입력이 비었을 때 "a"대입

6단계 : 15자 남기고 뒷 문자 제거(끝 문자가 마침표라면 제거)

7단계 : 2자 이하일 경우 마지막 문자를 반복해서 붙임
        


# 코드로 구현
```
import java.util.*;

class Solution {
    public String solution(String new_id) {
        // 1단계 : 대문자를 소문자로 치환
        String id = new_id.toLowerCase();

        // 2단계 : 허용 문자(소문자, 숫자, -, _, .) 외 제거
        id = id.replaceAll("[^a-z0-9\\-_.]", "");

        // 3단계 : 연속된 마침표를 마침표 하나로 치환
        id = id.replaceAll("\\.{2,}", ".");

        // 4단계 : 처음/끝 마침표 제거
        id = id.replaceAll("^\\.|\\.$", "");

        // 5단계 : 빈 문자열이면 "a" 대입
        if (id.isEmpty()) {
            id = "a";
        }

        // 6단계: 15자 남기고 뒷 문자 제거, 끝 마침표 제거
        if (id.length() >= 16) {
            id = id.substring(0, 15);
            id = id.replaceAll("\\.$", "");
        }

        // 7단계: 길이 2 이하일 경우 마지막 문자를 길이 3이 될 때까지 반복해서 붙임
        while (id.length() <= 2) {
            id += id.charAt(id.length() - 1);
        }

        return id;
    }
}
```

## 느낀점
한 단계씩 강의자료를 찾아가며 풀어내었다. 정규식을 글로 읽기만 했을 때보다 직접 활용해보니 더 쉽게 이해가 되었고, 헷갈리는 부분들도 숙지할 수 있었다.
